Instructions v0.1

  1. Open up this spreadsheet
  2. Find a courthouse that is missing a Map image filename
  3. Copy and paste the entire courthouse address into the Address input below the maps on this page and press Enter. The maps should now show markers at the address you entered. The top map is a google map, and the bottom is an OpenStreetMap.
  4. Copy and paste the court and courthouse names from the spreadsheet into the Filename input on this page. The naming convention we use is court-courthouse-name. Look at the Map image filename column in the spreadsheet for examples. Edit the name in the Filename input to match this convention. You can make adjustments if you feel like it. It is not vital to the function of the website that this convention be followed--it's just for our sanity.
  5. Click the Download image button and wait for several seconds for the image to download. Look on your computer to make sure you know where the images are being downloaded.
  6. Copy and paste the contents of the Filename input back into the spreadsheet in the Map image filename column.
  7. You completed a map! Repeat for more courthouses.
  8. When you've completed a bunch, upload them to this google drive folder
